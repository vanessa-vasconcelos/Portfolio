Esse foi um projeto fullstack que desenvolvi com base em um tutorial da Fernanda Kipper. A ideia era treinar a integração entre front-end e back-end usando tecnologias modernas. Com ele, é possível visualizar os pratos disponíveis e cadastrar novos via formulário.
Utilizei React com TypeScript no front e Spring Boot com Java no back, além de React Query e uma API RESTful simples com Spring Web e JPA.
