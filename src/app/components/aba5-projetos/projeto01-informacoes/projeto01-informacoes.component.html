Esse projeto foi uma ótima oportunidade para aprender, na prática, como funciona o fluxo completo de autenticação. Desenvolvi uma aplicação com cadastro e login de usuários usando Angular no front e integração com backend em Java com Spring Boot.
Implementei rotas protegidas, autenticação com JWT, formulários reativos e toda a lógica de verificação de credenciais.
